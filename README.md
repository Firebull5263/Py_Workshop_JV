# WeatherAI

An adaptive Streamlit application powered by XGBoost that automatically analyses uploaded datasets.

## Features

- Automatic dataset inspection
- Automatic target detection
- Automatic timestamp detection
- Automatic preprocessing
- Automatic XGBoost model selection
- Interactive Plotly visualisations
- Downloadable reports
- Works with unseen datasets

---

## Installation

```bash
git clone https://github.com/yourusername/weatherai

cd weatherai

pip install -r requirements.txt

streamlit run app.py
```

---

## Workflow

1 Upload CSV

2 Inspect Dataset

3 Automatic Preprocessing

4 Train Model

5 Evaluate Model

6 Visualise Results

7 Download Reports

---

## Deployment

Push to GitHub.

Connect the repository to Streamlit Community Cloud.

Deploy.
